Form Inputs

Data is updated on input onChange event so will happen when tabbing away from the field. Checkboxes expect a boolean value. Radio buttons are finicky and need to use data-click to set their value.

Javascript

const person = new Component('form', {
	data: {
		firstName: 'Stoo',
		lastName: 'Goff',
		age: 48,
		hobby: '',
		checked: true,
		radio: 'Programming',
	},
	setRadio(evt) {
		this.data.radio = evt.target.value
	}
})
